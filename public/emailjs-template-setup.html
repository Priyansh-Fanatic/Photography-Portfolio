EmailJS Template Setup Guide

This guide will help you set up your EmailJS template correctly to ensure all form fields are sent in the email.

EmailJS Template Example

Create a new email template with the following template variables:

From: {{from_name}} <{{from_email}}>
Reply-To: {{reply_to}}
Subject: Contact Form: {{subject}}

NEW CONTACT FORM SUBMISSION

FROM: {{name}} <{{email}}>
SUBJECT: {{subject}}
TIME: {{timestamp}}

MESSAGE:
{{message}}

---
This message was sent from your photography portfolio website contact form.

Alternative: Use the Formatted Message

If you prefer, you can use the pre-formatted message that includes all details:

{{formatted_message}}
Note: Make sure to update the EmailJS service ID and template ID in the contact form component.
